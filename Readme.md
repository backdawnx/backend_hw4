## 📌 Project Description
โปรแกรมประมวลผลและแสดงข้อมูลจากไฟล์ CSV (`malware_500.csv`) ที่พัฒนาด้วย Windows Forms (.NET) โดยมีฟีเจอร์หลักในการจัดการข้อมูลดังนี้:
- **Range Loading:** สามารถจำกัดช่วงบรรทัดที่ต้องการอ่านและแสดงผลได้ (ระบุช่วงแถว m ถึง n)
- **Data Filtering:** สามารถกรองข้อมูลตามประเภทของไฟล์ (File Extension) เช่น `elf`, `exe`, `sh` หรือแสดงผลทั้งหมด (`All`)
- **Error Handling:** มีระบบตรวจสอบความถูกต้องของข้อมูล (Input Validation) เช่น แจ้งเตือนเมื่อกรอกตัวอักษรลงในช่องตัวเลข หรือเมื่อระบุช่วงบรรทัดขัดแย้งกัน

## 🚀 How to Use
1. เปิดโปรแกรมและคลิกปุ่ม `Browse` เพื่อเลือกไฟล์ `malware_500.csv`
2. กดเลือก Tab ชื่อ CSV
3. ระบุช่วงข้อมูลที่ต้องการในช่อง **StartLine** และ **stopLine** (ต้องเป็นตัวเลขเท่านั้น)
4. เลือกประเภทไฟล์ที่ต้องการกรองในช่อง **File Extension**
5. คลิกปุ่ม `Search` เพื่อแสดงผลข้อมูลลงในตาราง DataGrid
